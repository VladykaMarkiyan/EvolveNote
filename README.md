# 📝 EvolveNote — Fullstack Project Setup Guide

---

## 📱 1. Фронтенд (EvolveNoteMobile)

1. Створити .env в корені EvolveNoteMobile
2. Вставити у змінну свій IPv4 Address як у прикладі в .env.example
3. Бібліотека: 
npx expo install expo-router react-native-safe-area-context react-native-screens
4. Запуск: npx expo start


## 📱 2. Бекенд (backend)

1. Бібліотека: 
pip install flask flask-sqlalchemy flask-cors werkzeug
2. Запуск: python app.py
